# Lifeline Mobile Application

A monetized content platform that transforms personal digital data into compelling, AI-generated biographies.

## Project Structure

```
lifeline/
├── mobile/           # React Native mobile app
├── backend/          # Node.js API server
└── Build.md         # Development roadmap
```

## Getting Started

### Prerequisites
- Node.js >= 18
- npm or yarn
- Expo Go app (for mobile testing)

### Mobile App
```bash
cd mobile
npm install
npm start
```

### Backend API
```bash
cd backend
npm install
npm run dev
```

## Tech Stack

### Mobile
- React Native with Expo
- TypeScript
- React Navigation
- AsyncStorage
- Axios

### Backend
- Node.js with Express
- TypeScript
- PostgreSQL
- JWT Authentication
- Prisma ORM
